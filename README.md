## 简介

**基于聚创API数据的生活服务网站**

## 项目结构
- vue-demo (前端主程序)
- myapp (后端程序)
- env (python 虚拟环境)

## 如何使用
- 启动Flask服务 
        *使用5000端口*
    ```sh 
    $ cd myapp
    $ python routes.py
    ```
- 开启前端服务
        *使用8085端口*
    ```sh
    $ npm run dev
    ```
    *如何端口开启失败可以运行 $npm install*
